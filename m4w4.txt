Type of CSS selectors

Selector Type / Example / Selects

Universal Selector * All elements
Type Selector p, h1, div All <p>, <h1>, or <div> elements
Class Selector .box All elements with class="box"
ID Selector #header The element with id="header"
Group Selector h1, p, a All h1, p. and a elements
Descendant Selector div p All <p> inside <div>
Attribute Selector input[type="text] All input fields with type="text"
Pseudo-class a:hover,li:first-child Elements in a specific state
Pseudo-element p::first-line Specific part of an element
